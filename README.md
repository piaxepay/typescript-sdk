# Piaxis SDKs

This workspace holds language-specific SDKs for the public partner API served from `/api`.

- `typescript/`: JavaScript/TypeScript SDK
- `python/`: Python SDK
- `contracts/`: shared contract fixtures used by both SDK test suites

Architecture and sandbox guidance lives in `ARCHITECTURE.md`.
Publishing instructions live in `DEPLOYMENT_GUIDE.md`.
Partner onboarding lives in `SANDBOX_ONBOARDING.md`.
